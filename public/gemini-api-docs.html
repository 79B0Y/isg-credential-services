📋 概述

Google Gemini API提供了强大的AI聊天功能，支持多种模型和参数配置。本服务集成了Google Gemini的官方API，提供简单易用的接口。

🔑 主要功能

  • 模型管理: 查询可用的Gemini模型列表
  • 聊天对话: 支持多轮对话和复杂消息结构
  • 简单聊天: 系统提示词 + 用户提示词的简化接口
  • 参数配置: 支持温度、最大令牌数等参数调整
  • 安全设置: 内置安全过滤和内容审核
  • 使用统计: 显示Token使用情况和成本信息

📊 支持的模型

  • Gemini 2.5 Pro: 最强大的模型，擅长推理、多模态理解、高级编程等复杂任务
  • Gemini 2.5 Flash: 快速且成本效益高，适合需要快速响应和高吞吐量的任务（推荐）
  • Gemini 2.5 Flash-Lite: 最经济实惠的价格，适合简单任务的高吞吐量
  • Gemini 2.0 Flash: 快速响应的模型
  • Gemini 1.5 Pro: 强大的专业模型
  • Gemini 1.5 Flash: 快速响应的轻量级模型
  • Gemini Pro: 经典的文本生成模型
  • Gemini Pro Vision: 支持图像理解的模型

🤖 模型管理

GET /api/gemini/gemini/models
获取可用的Gemini模型列表
curl -X GET http://localhost:3000/api/gemini/gemini/models

💬 聊天对话

POST /api/gemini/gemini/chat
发送聊天消息到Gemini，支持多轮对话
curl -X POST http://localhost:3000/api/gemini/gemini/chat \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user", "content": "你好，请介绍一下自己"}
    ],
    "options": {
      "model": "gemini-2.5-flash",
      "temperature": 0.7,
      "max_tokens": 1000
    }
  }'

📋 请求参数

{
  "messages": [
    {
      "role": "user",       // 用户角色
      "content": "用户消息"
    },
    {
      "role": "model",      // 模型角色
      "content": "模型回复"
    }
  ],
  "options": {
    "model": "gemini-pro",              // 模型名称
    "temperature": 0.7,                 // 温度 (0-2)
    "max_tokens": 1000,                 // 最大令牌数
    "top_p": 1,                         // Top-p采样
    "top_k": 40,                        // Top-k采样
    "safety_settings": true             // 安全设置
  }
}

💭 简单聊天

POST /api/gemini/gemini/simple-chat
简化的聊天接口，支持系统提示词和用户提示词
curl -X POST http://localhost:3000/api/gemini/gemini/simple-chat \
  -H "Content-Type: application/json" \
  -d '{
    "system_prompt": "你是一个有用的AI助手",
    "user_prompt": "请介绍一下Google Gemini",
    "options": {
      "model": "gemini-2.5-flash",
      "temperature": 0.7,
      "max_tokens": 1000
    }
  }'

📋 请求参数

{
  "system_prompt": "你是一个有用的AI助手",  // 系统提示词（可选）
  "user_prompt": "请介绍一下Google Gemini", // 用户提示词（可选）
  "options": {
    "model": "gemini-pro",              // 模型名称
    "temperature": 0.7,                 // 温度
    "max_tokens": 1000                  // 最大令牌数
  }
}
