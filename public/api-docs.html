📱 Telegram 消息管理

GET /api/telegram/telegram/messages
获取消息历史
curl -X GET http://localhost:3000/api/telegram/telegram/messages
POST /api/telegram/telegram/polling/start
开始消息轮询
curl -X POST http://localhost:3000/api/telegram/telegram/polling/start
POST /api/telegram/telegram/send/message
发送文本消息
curl -X POST http://localhost:3000/api/telegram/telegram/send/message \
  -H "Content-Type: application/json" \
  -d '{"chat_id": "123456789", "text": "Hello World"}'
POST /api/telegram/telegram/send/photo
发送图片消息
curl -X POST http://localhost:3000/api/telegram/telegram/send/photo \
  -H "Content-Type: application/json" \
  -d '{"chat_id": "123456789", "photo": "https://example.com/image.jpg", "caption": "Photo caption"}'

🔗 Webhook 管理

POST /api/telegram/telegram/webhook/set
设置Webhook URL
curl -X POST http://localhost:3000/api/telegram/telegram/webhook/set \
  -H "Content-Type: application/json" \
  -d '{"url": "https://your-domain.com/webhook"}'
DELETE /api/telegram/telegram/webhook
删除Webhook
curl -X DELETE http://localhost:3000/api/telegram/telegram/webhook

🌐 WebSocket 管理

POST /api/telegram/telegram/websocket/start
启动WebSocket服务器，用于实时接收Telegram消息
curl -X POST http://localhost:3000/api/telegram/telegram/websocket/start \
  -H "Content-Type: application/json" \
  -d '{"port": 8080}'
参数:
  • port (可选): WebSocket端口，默认为8080
POST /api/telegram/telegram/websocket/stop
停止WebSocket服务器
curl -X POST http://localhost:3000/api/telegram/telegram/websocket/stop
GET /api/telegram/telegram/websocket/status
获取WebSocket服务器状态
curl -X GET http://localhost:3000/api/telegram/telegram/websocket/status
响应示例: 
{
  "success": true,
  "data": {
    "running": true,
    "port": 8080,
    "clients": 2,
    "url": "ws://localhost:8080"
  }
}

📡 WebSocket 连接说明

WebSocket服务器启动后，可以通过以下方式连接：

  • 连接URL: ws://localhost:8080
  • 消息格式: JSON
  • 支持的消息类型:
    • get_status - 获取状态
    • get_messages - 获取消息历史
    • start_polling - 开始轮询
    • stop_polling - 停止轮询

JavaScript 连接示例:

const ws = new WebSocket('ws://localhost:8080');

ws.onopen = function() {
    console.log('WebSocket连接已建立');
    // 获取状态
    ws.send(JSON.stringify({type: 'get_status'}));
};

ws.onmessage = function(event) {
    const data = JSON.parse(event.data);
    console.log('收到消息:', data);
    
    if (data.type === 'new_message') {
        console.log('新消息:', data.data);
    }
};

ws.onclose = function() {
    console.log('WebSocket连接已关闭');
};

Node.js 连接示例:

const WebSocket = require('ws');

const ws = new WebSocket('ws://localhost:8080');

ws.on('open', function() {
    console.log('WebSocket连接已建立');
    // 获取状态
    ws.send(JSON.stringify({type: 'get_status'}));
});

ws.on('message', function(data) {
    const message = JSON.parse(data);
    console.log('收到消息:', message);
    
    if (message.type === 'new_message') {
        console.log('新消息:', message.data);
    }
});

ws.on('close', function() {
    console.log('WebSocket连接已关闭');
});
